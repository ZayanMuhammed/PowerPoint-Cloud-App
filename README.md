# PowerPoint Cloud App

PowerPoint Cloud App is a desktop application built with **Electron.js** that provides access to **Microsoft PowerPoint Online** through a dedicated app interface. It acts as a lightweight wrapper around the official Microsoft 365 web experience, allowing users to work with PowerPoint presentations without opening a traditional browser.

* Access Microsoft PowerPoint Online in a desktop app
* Lightweight and fast Electron-based wrapper
* No local installation of Microsoft PowerPoint required
* Uses the official Microsoft 365 web interface
* Cross-platform support (Windows, macOS, Linux)
